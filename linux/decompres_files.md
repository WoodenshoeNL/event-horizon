#Decompres Files

## Decompress gzip file  (tgz)

```bash
tar -zxvf <filename>
```

## Decompress bzip file  (tbz)

```bash
tar -jxvf <filename>
```

## Decompress bz2 file

```bash
bunzip2 <filename>
```
