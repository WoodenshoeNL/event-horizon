
# Installing Ghidra on Kali

Install OpenJDK first

```sh
sudo apt install default-jdk
```



[Download Ghidra](https://www.ghidra-sre.org/)


Unzip Ghidra

```sh
cd ~/Downloads
unzip ghidra_*_PUBLIC_*.zip
```

Go into the Ghidra directory and run **ghidraRun**

```sh
cd ghidra_*_PUBLIC
./ghidraRun
```
