
Start gdh with a program

```bash
gdh ./hackme
```

Run program with the letter r

```bash
r
```

Disassemble a function

```bash
disas main
```


Set Break point Function name
```bash
break main
```

Set Break point address
```bash
break *0x00000000000012a5
```

Get info break points
```bash
info break
```

Delete breakpoint by number
```bash
d <breakpoint number>
```
